# CSRC基金报告搜索参数测试模板

## 📋 测试准备

### 网站地址
```
http://eid.csrc.gov.cn/fund/disclose/advanced_search_xbrl.do
```

### 操作步骤
1. 打开上述网站
2. 按F12打开开发者工具
3. 切换到 **Network** 标签
4. 清空网络请求记录
5. 按照下面的测试用例逐一操作
6. 每次点击"查询"后，查找包含 `advanced_search_xbrl.do` 的请求
7. 点击该请求，查看 **Request URL** 中的 `aoData` 参数
8. 将参数值填入对应的表格中

---
